So I’m setting up ampache 3.7.0 on a local server, and here are some tips I’ve found useful:

  • There are a bunch of useful CLI tools shipped with the source. Use them! Like bin/install/add_user.inc.

  • The debugging log is very handy, and details a lot if internal information. Wondering why that flac doesn’t play? Look there. Here are my settings: (from config/ampache.cfg.php)

    debug = true debuglevel = 5 logpath = “/srv/ampache/logs” log_filename = “%name.%Y%m%d.log”

Though granted, it doesn’t do the best log rotation. You’ll need to clean out old log files.

Speaking of FLAC playing, the HTML5 player that Ampache ships with is jPlayer. Which doesn’t support FLAC. So let’s enable transcoding it to OGG:

transcode_flac = required
encode_target_flac = ogg
transcode_cmd = "ffmpg -i %FILE%"
encode_args_ogg = "-vn -b:a %SAMPLE%K -c:a libvorbis -f ogg pipe:1"
